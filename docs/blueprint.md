# **App Name**: Viña Negra Manager

## Core Features:

- Contact Management: Create, read, update, and delete client and supplier contacts, storing key information like RUT, address, and contact person.
- Sales Order Generation: Generate sales orders, importing items directly from purchase orders to streamline the process.
- Financial Summary Generation: Use the Gemini API tool to generate executive summaries of the financial status and suggest descriptions for financial transactions.
- Purchase Order Generation: Generate purchase orders for acquired products, with standardized fields (dropdown lists) for data consistency.
- Service Order Management: Record operational costs like freight or labor, linking each service order to a purchase order for cost control.
- Financial Movement Recording: Log all income and expenses, associating them with sales, purchase, or service orders for financial traceability.
- Dashboard Visualization: Visualize KPIs (Kilos bought/sold, revenue, expenses, service costs) and interactive graphs (weekly revenue, expense breakdown, kilo comparison, caliber distribution).

## Style Guidelines:

- Primary color: Soft green (#A7D1AB), evoking nature, freshness, and growth, complementing the themes of agriculture and business.
- Background color: Light, desaturated green (#E8F2E9) provides a clean, professional backdrop, ensuring readability and reducing visual fatigue.
- Accent color: Muted blue (#8FB3D1), an analogous color used for interactive elements and key highlights to guide user attention subtly.
- Headline font: 'Belleza', a humanist sans-serif, adding personality, and a style aligned to fashion, art, and design.
- Body font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel.
- Use clean, professional icons representing each module, such as a truck for service orders or a graph for the dashboard.
- Implement a fixed sidebar for desktop views for quick access to all modules, ensuring a professional and efficient user experience.
